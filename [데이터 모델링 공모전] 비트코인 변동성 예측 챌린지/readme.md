# BDA 채용 연계 데이터 모델링 공모전 
- **배경** : 실제 트레이딩 펌에서 사용하는 시계열 데이터를 직접 다뤄봄으로써 데이터 핸들링 숙련도를 평가하고 퀀트 트레이딩 분야에 대한 도전 기회 제공
- **대회 목적** :
- 데이터 : Tick data 중 체결 데이터 (Trad Data)
- 평가 산식 : MAPE
- 활용 코드 : 주식 종가 예측 대회의 아이디어에서 착안해서 주 단위 예측 -> 일 단위 예측 진행
